
# Sistema de Gestión de Libros Electrónicos

Proyecto académico correspondiente a la **Etapa 1: Planeación del Software**.

Este sistema tiene como objetivo administrar libros electrónicos de manera organizada, aplicando los principios de la **programación funcional** utilizando el lenguaje **Go (Golang)**.

## Objetivo del proyecto

Desarrollar un sistema que permita:
- Registrar libros electrónicos
- Consultar y listar libros
- Gestionar información básica de usuarios
- Organizar el código mediante paquetes y funciones


## Tecnologías utilizadas

- Go (Golang)
- Git
- GitHub

## Paradigma de programación

El sistema se desarrolla utilizando **programación funcional**, mediante:
- Funciones puras
- Retorno de valores
- Uso mínimo de variables globales
- Separación de la lógica en funciones independientes
